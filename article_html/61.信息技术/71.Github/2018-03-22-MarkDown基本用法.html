h1

3.引用

> aaaaaaaaa
>> bbbbbbbbb

4.行内标记

`
<div>   
<div></div>
</div>
`

注：标记内的文本会被转换为一行

5.代码块

5.1 用(Tab)缩进

5.2 用(```)注释

```
Code
```

5.3 自定义语法

```javascript
var num = 0;
for (var i=0; i<5; i++){
    num+=i;}
console.log(num);
```

6.插入链接

6.1 内链式

[百度](http://www.baidu.com/){:target="_blank"}

6.2 引用式

[百度][2]{target="_blank"}
[2]:http://www.baidu.com/

7.插入图片

7.1 内链式

![name](./01.png '描述')

7.2 引用式

![name][01]
[01]: ./01.png '描述'

8.插入图片带有链接

8.1 内链式

[![](./01.png '百度')](http://www.baidu.com){:target="_blank"}

8.2 引用式

[![](./01.png '百度')][5]{:target="_blank"}
[5]: http://www.baidu.com
