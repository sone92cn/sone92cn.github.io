PyMySQL基本用法

导入库

import pymysql

创建连接

conn = pymysql.connect(host='127.0.0.1', port=3306, user='root', passwd='', db='tkq1', charset='utf8')

创建游标

cursor = conn.cursor()

执行SQL，并返回收影响行数

effect_row = cursor.execute("select * from tb7")

执行SQL，并返回受影响行数

effect_row = cursor.execute("update tb7 set pass = '123' where nid = %s", (11,))

执行SQL，并返回受影响行数,执行多次
