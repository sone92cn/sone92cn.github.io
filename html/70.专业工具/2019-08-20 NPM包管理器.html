NPM包管理器

一、npm安装

下载并安装node.js，会自动安装npm

二、npm设置

1. 修改包安装目录：

在nodejs根目录下，node_modules/npm文件夹中找到npmrc文件，修改其中prefix为新路径，如：E:\Projects\npm

三、安装包

1. 当前目录安装：

npm install packagename@v，省略@v将自动安装最新版本，比如：

npm install jquery@3

2. 全局安装：

npm install packagename@v -g，将安装到第二步设置的目录。

四、查看已安装的包

npm ls [-g]
