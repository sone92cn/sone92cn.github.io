Atom插件管理

一、插件安装方法

  1. Setting->Install中安装
  2. apm方式：apm install packagename
  3. 已安装插件位置：Users\User.atom\packages
    （将插件文件复制到以上目录，会被自动安装）

二、Markdown插件

  1. 安装language-markdown插件（或者启用默认的language-gfm插件）
  2. 安装markdown-writer插件
  3. 查看markdown-preview插件是否安装
  4. 为了得到更好的预览体验可以安装markdown-preview-plus插件，启动速度会更慢
    markdown-preview-plus样式配置文件
  5. "Users\User\.atom\packages\markdown-preview-plus\styles-client\default.less"
  6. markdown-img-paste图片粘贴
  7. markdown-table-editor表格编辑

三、Python插件

  1. 代码高亮：Atom自带
  2. 自动补全：autocomplete-python
  3. 语法检查：linter-flake8
    Python需安装flake8
  4. 定义跳转：python-tools
  5. 代码运行：
    atom-runner （只能输出，不能输入）
    atom-python-run（Windows，可以输入）
    python-run-terminalnx（Linux，可以输入）
  6. 在linter-flake8的设置中可以取消一行79个字符的限制
  7. highlight-selected 双击变量以追踪

四、其它插件

  1. minimap: 代码小地图
  2. atom-beautify：美化代码格式(ctrl + alt + b)
  3. script: 代码调试（调用python乱码，添加环境变量PYTHONIOENCODEING=UTF-8）
  4. platformio-ide-terminal：在界面下方添加终端
  5. Hydrogen
