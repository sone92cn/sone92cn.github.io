创建虚拟环境

pip install virtualenv pip install virtualenvwrapper-win 配置环境变量：WORKON_HOME=path 创建：mkvirtualenv venvname 查看：workon 启动：workon venvname 退出：deactivate 删除：rmvirtualenv venv

启动虚拟环境

切换到Virtualenv\superset\Scripts目录，运行"activate"

安装superset

pip install superset (如部分依赖库安装失败，手工下载安装)

初始化superset

进入 Superset 安装目录（ ...\Anaconda3\envs\superset\Lib\site-packages\superset\bin） python superset db upgrade # 初始化数据库 python superset load_examples # 载入案例数据 python superset init # 初始化角色和权限 python superset runserver # 启动服务，端口号 8088，使用 -p 更改端口号 python superset runserver -d # Windows需要在命令行中添加-d，使用 development web server 运行，不推荐正式环境使用 python superset run # python37 win10 尝试这个命令

进入虚拟环境
