MongoDB安装

一、下载MongoDB(Zip版)

url: https://fastdl.mongodb.org/win32/mongodb-win32-x86_64-2012plus-4.2.2.zip

二、解压到任意目录

如：D:\MongoDB（若无logs目录，在跟目录下新建）

三、添加根目录下"bin"到系统环境变量

四、在任意位置创建MongoDB配置文件

如：E:\MongoDB\Conf\mongo.config

Superset安装

一、创建虚拟环境

  • pip install virtualenv
  • pip install virtualenvwrapper-win
  • 配置环境变量：WORKON_HOME=path
  • 创建：mkvirtualenv venvname
  • 查看：workon
  • 启动：workon venvname
  • 退出：deactivate
  • 删除：rmvirtualenv venv

二、启动虚拟环境

切换到Virtualenv\superset\Scripts目录，运行"activate"
