如何安全地修改密码：

方式1 使用psql，连接到Postgres Server：

test1=> \password Enter new password: Enter it again: test1=> 我将原密码hello，修改为hellojava.123456 这种修改方式相当于向postgres server 发送了如下命令：

ALTER USER postgres PASSWORD ' md53175af154as54df5as4d5f45sd6af'; 后面的字符串是 hellojava经过md5加密后的字符串 12345

MongoDB安装

一、下载MongoDB(Zip版)

url: https://fastdl.mongodb.org/win32/mongodb-win32-x86_64-2012plus-4.2.2.zip

二、解压到任意目录

如：D:\MongoDB（若无logs目录，在跟目录下新建）

三、添加根目录下"bin"到系统环境变量

四、在任意位置创建MongoDB配置文件

如：E:\MongoDB\Conf\mongo.config

创建虚拟环境

pip install virtualenv pip install virtualenvwrapper-win 配置环境变量：WORKON_HOME=path 创建：mkvirtualenv venvname 查看：workon 启动：workon venvname 退出：deactivate 删除：rmvirtualenv venv

切换到Virtualenv\superset\Scripts目录，运行"activate"

CMD编码修改

  1. 临时修改：只作用于当前窗口，先进入cmd命令窗口（快捷键win键+R）
  2. 直接输入“chcp 65001”，回车键（Enter键）执行，这时候该窗口编码已经是UTF-8编码了。
  3. 永久修改：win键+R，输入regedit，确定。
  4. 按顺序找到HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Command Processor。
  5. 点击右键-新建，选择“字符串值”。
  6. 命名为“autorun”, 点击右击修改，数值数据填写“chcp 65001”，确定。
